## exists

```PHP
判断是否存在
  method_exists() — 检查类的方法是否存在
  class_exists() - 检查类是否已被定义
  file_exists() 函数检查文件或目录是否存在。
```

## strpos

```
strpos() 函数返回字符串在另一个字符串中第一次出现的位置。
	如果没有找到该字符串，则返回 false。
strpos(string,find,start) 　　参数 描述
	string 必需。规定被搜索的字符串。
	find 必需。规定要查找的字符。
	start 可选。规定开始搜索的位置。
	
```
