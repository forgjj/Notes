## PHP date

```
PHP date() 函数用于对日期或时间进行格式化。
PHP Date() 函数把时间戳格式化为更易读的日期和时间。
d - 表示月里的某天（01-31）
m - 表示月（01-12）
Y - 表示年（四位数）
1 - 表示周里的某天
```
